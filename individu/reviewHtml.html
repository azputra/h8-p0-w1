Kegiatan di Hari pertama kami adalah perkenalan dengan teman teman yang akan berjuang bersama selama di hacktiv8 dan kami di berikan breefing oleh instruktur kami tentang kegiatan kami selama berada di bootcamp

Materi Apa Yang Di Beri Di?

Yup . .
Seperti yang kalian ketahui kalau jadwal di hacktiv itu sangat padat di hari pertama saja kami sudah di beri berbagai macam intruksi yaitu :

  • Mengunjungi Foxhub: yang merupakan salah satu web hacktiv8 yang dimana fungsinya untuk mengumpulkan tugas kami saat berada di phase 0
  • Membuka Materi Di Hacktiv Den: yang merupakan salah satu web hacktiv8 yang dimana fungsinya adalah kumpulan dari materi pembelajaran dan tugas-tugas phase 0
  • Install Slack: yang merupakan media komunikasi dan file sharing
  • Daftar Github: yang merupakan software version control platform
  • Daftar codecademy: yang merupakan Platform untuk enroll kelas atau belajar programing basic
  • Install Nodejs: Untuk mengeksekusi code javascript di luar browser
  • Repl dan JSBin: Online IDE a tool for experimenting with web languages
  • Mengerjakan html dan css mengerjakan tugas HTML dan CSS yang ada di codecademy
